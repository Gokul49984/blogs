Blogs Collection using Node JS.
----------------------------
REST APIs to CREATE, VIEW, UPDATE and DELETE Blogs.


Usage:
------
-> Run npm install in your root project folder.

-> Run npm start to start the server (Default Port: 3000. It can be modified in the env.config.js file).

-> Import the postman collection and send requests.

-> Access Token returned from the login API will be automatically inherited by other APIs using environmental variables.
